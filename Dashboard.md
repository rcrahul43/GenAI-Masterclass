## Start a session

1. Open this week in [Study Plan](Study%20Plan.md) (or set Week # on the Excel Dashboard).
2. Open the next module (sheet **Next 5** or [Progress Tracker](Progress%20Tracker.md)).
3. Study that one module only.
4. Mark it Done in the Excel **Modules** sheet when lab + judgment note are finished.
